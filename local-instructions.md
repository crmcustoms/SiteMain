# Инструкция для локального запуска диагностики

## Что нужно сделать на вашем компьютере

1. **Откройте PowerShell в папке проекта:**
   ```
   cd h:\Cursor\SiteMain
   ```

2. **Запустите диагностический скрипт:**
   ```
   .\step1-check-current-state.ps1
   ```

3. **Скопируйте результат** и отправьте его мне

## Что проверяет скрипт

- Состояние Git репозитория
- Структуру файлов проекта  
- Настройки Next.js
- Docker файлы (если есть)
- Переменные окружения

## Если возникли ошибки

Если появится ошибка "execution policy", выполните:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Затем повторите запуск скрипта.

---

**Важно:** Этот скрипт запускается ТОЛЬКО на вашем локальном компьютере, не в Docker!
