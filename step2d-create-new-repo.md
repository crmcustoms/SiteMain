# Step 2D: Создание нового безопасного репозитория

## Что делаем:
Создаем новый GitHub репозиторий с чистой историей (без секретов).

## Шаги:

### 1. Удаление старого репозитория
1. Откройте https://github.com/crmcustoms/SiteMain
2. Перейдите в Settings (внизу страницы)
3. Прокрутите вниз до "Danger Zone"
4. Нажмите "Delete this repository"
5. Подтвердите удаление

### 2. Создание нового репозитория
1. Зайдите на https://github.com
2. Нажмите зеленую кнопку "New"
3. Repository name: `SiteMain`
4. Description: `Next.js website with Notion CMS and secure deployment`
5. ❌ НЕ ставьте галочки на README, .gitignore, license
6. Нажмите "Create repository"

### 3. Подготовка локального репозитория
После создания нового репозитория я помогу:
- Удалить старый remote
- Добавить новый remote
- Загрузить код с исправленными секретами

## Статус:
✅ Секреты убраны из кода
✅ Переменные окружения настроены
⏳ Ждем создания нового репозитория
