# Step 2A: Создание нового GitHub репозитория

## Что мы делаем:
Создаем новый GitHub репозиторий и подключаем к нему ваш локальный код.

## Вариант выполнения:

### Вариант А: Через веб-интерфейс GitHub (проще)
1. Зайдите на https://github.com
2. Нажмите зеленую кнопку "New" (или "+" → "New repository")
3. Заполните:
   - **Repository name**: `SiteMain` (или другое название)
   - **Description**: `Next.js website with blog and multilingual support`
   - **Public/Private**: выберите приватность
   - ❌ НЕ ставьте галочки на README, .gitignore, license (у нас уже есть код)
4. Нажмите "Create repository"
5. Скопируйте URL репозитория (будет показан на следующей странице)

### Вариант Б: Через GitHub CLI (если установлен)
Я могу помочь с командами, если у вас установлен GitHub CLI.

## Следующий шаг:
После создания репозитория сообщите мне:
1. URL созданного репозитория
2. Какое название вы выбрали

Тогда переходим к **Step 2B: Подключение локального кода к GitHub**
