# Проект: Сайт для продажи услуг по автоматизации бизнеса

## Общая информация
**Название:** SiteMain  
**Цель:** Создание персонального сайта для продажи услуг по автоматизации бизнеса  
**Статус:** В разработке (приблизительно 50% завершено)  

## Архитектура проекта

### Технологический стек
- **Фреймворк:** Next.js 15
- **Языки программирования:** TypeScript, JavaScript
- **Стилизация:** Tailwind CSS
- **UI компоненты:** Radix UI
- **Формы:** React Hook Form с валидацией Zod
- **CMS/База знаний:** Интеграция с Notion API
- **Темизация:** next-themes

### Структура проекта
```
SiteMain/
├── app/                    # Основная директория приложения Next.js App Router
│   ├── landing/            # Компоненты главной страницы
│   ├── cases/              # Страница кейсов/проектов
│   ├── [lang]/             # Мультиязычность
│   ├── api/                # API endpoints
│   ├── layout.tsx          # Корневой макет приложения
│   ├── page.tsx            # Главная страница
│   └── globals.css         # Глобальные стили
├── components/             # Переиспользуемые компоненты
├── lib/                    # Утилиты и общие функции
├── hooks/                  # React хуки
├── styles/                 # Дополнительные стили
├── public/                 # Статические файлы
├── favicons/               # Фавиконки
└── docs/                   # Документация проекта
```

### Основные компоненты и функциональные модули
1. **Лендинг** - Презентация услуг автоматизации
2. **Кейсы** - Портфолио реализованных проектов
3. **Интеграция с Notion API** - Для управления контентом
4. **Мультиязычность** - Поддержка различных языков
5. **Формы обратной связи** - Для приёма заявок клиентов

## Требования и функциональность

### Функциональные требования
1. **Презентация услуг:**
   - Детальное описание предлагаемых услуг автоматизации
   - Преимущества и выгоды для клиента
   
2. **Демонстрация экспертизы:**
   - Портфолио реализованных проектов
   - Кейсы с описанием решенных проблем
   
3. **Взаимодействие с клиентом:**
   - Форма обратной связи
   - Контактная информация
   - Конверсионные элементы (CTA)
   
4. **Контентная составляющая:**
   - Блог с полезными материалами
   - Интеграция контента из Notion

### Нефункциональные требования
1. **Производительность:**
   - Оптимизация Core Web Vitals
   - Быстрая загрузка страниц (< 2 секунд)
   
2. **Безопасность:**
   - Защита форм от спама и атак
   - Безопасное хранение данных клиентов
   
3. **SEO-оптимизация:**
   - Структурированная семантическая разметка
   - Оптимизация метаданных
   - Sitemap и robots.txt

4. **Доступность:**
   - Соответствие WCAG стандартам
   - Поддержка скринридеров
   
5. **Адаптивный дизайн:**
   - Корректное отображение на всех устройствах
   - Mobile-first подход
